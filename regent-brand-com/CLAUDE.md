# Regent Brand.com Redesign

## The Task
Rebuild regent.com as a Brand.com site using IHG's Wilson design system. This is a 1:1 content migration — same content as the live site today, new components and design system applied. No dynamic CMS content; this is authored content.

## Pages in Scope
- Main homepage
- Hotels & resorts pages
- Residences section (apartment purchasing info — essentially a brochure, links out externally)

## Design Goals
- Apply Wilson design system components to existing content
- Match the elevated, luxury feel of the Regent brand
- Identify content gaps where existing components don't fit and propose net-new components (must be reusable)
- Explore animation/interaction ideas to elevate the brand experience
- Reference the existing regent.com live site for content
